## Getting started

-   Download the **[Extension VSIX](https://microsoft.github.io/data-science-editor/vscode/data-science-editor.vsix)**
-   In Visual Studio Code, open the command palette and run **Extensions: Install from VSIX...** and select the downloaded vsix.
-   Following the instruction in [About](./about)

## Workspace blocks

The **.CSV**, **.TSV** files can be accessed through the workspace blocks.

## Saving

The current default is to support a single file `data.dsejson` in the root folder of the workspace.
