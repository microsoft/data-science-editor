# Data Streamer

The [Data Streamer Excel Addon](https://www.microsoft.com/en-us/education/hackingstem/datastreamer) allows to import live sensor
data into your Excel spreadsheets. You can use Jacdac to create programs that will upload data into the Data Streamer.

## Settings up your project

-   Add the **data streamer** extension
-   Add the **jacdac** extension
-   connect all your sensors or start simulators
-   click on **Add Blocks** in the Jacdac dashboard to load the blocks

## Streaming data

In a `forever` loop, drag the `write numbers` block from the **data streamer** category
and add the sensor readinds into the array. This code will write data that the Data Streamer addon can import into your spreadsheet.

```blocks
dep: github:microsoft/pxt-data-streamer
-----
basic.forever(function () {
    led.toggle(0, 0)
    dataStreamer.writeNumberArray([
        modules.thermometer1.temperature(),
        modules.humidity1.humidity(),
        modules.barometer1.pressure(),
        modules.lightLevel1.lightLevel()
    ])
    dataStreamer.writeLine()
    pause(100)
})
```

Once you have downloaded this program to your micro:bit V2, you can go back to Excel and start a Data Streamer worksheet.
