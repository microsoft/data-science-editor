# LED

The LED pixel extension provides methods for working with a fixed, small strip of programmable RGB LEDs. 
For module with a large or variable number of LEDs, see the [LED strip](/clients/makecode/extensions/ledstrip/) service.

## Direct control of pixels

All the pixels can be set to a color using the `setAll` method:

```blocks
input.onButtonPressed(Button.A, () => {
    modules.led1.setAll(0x00ff00)
})
input.onButtonPressed(Button.B, () => {
    modules.led1.setAll(0xff0000)
})
```

The brightness can be set and retrieved as follows:

```blocks
input.onButtonPressed(Button.A, () => {
    modules.led1.setBrightness(85)
    let brightness = modules.led1.brightness()
    basic.showNumber(brightness)
})
```

Pixels can be individually set using the `setPixel` method as shown in the
example below, which also uses the `numPixels` method to get the number
of pixels supported by the underlying hardware.

```blocks
input.onButtonPressed(Button.A, () => {
    modules.led1.setAll(0)
    let n = modules.led1.numPixels()
    for(let i = 0; i <= n - 1; i++) {
        modules.led1.setPixelColor(i, 0x00ff00)
        basic.pause(200)
    }
})
```
