# Magnetic field level

The following event handlers are invoked when the magnetic field transitions
from low to high (active) or high to low (inactive):

```blocks
modules.magneticFieldLevel1.onActive(() => {
    led.stopAnimation()
    basic.showString("A")
})

modules.magneticFieldLevel1.onInactive(() => {
    led.stopAnimation()
    basic.showString("I")
})

```

The magnetic field level extension reports magnetic field strength as a floating
point value between -1.0 and 1.0 The `strength` method returns the current strength.

The following program outputs the magnetic strength whenever it changes by 0.22:

```blocks
modules.magneticFieldLevel1.onStrengthChangedBy(0.2, () => {
    led.plotBarGraph(modules.magneticFieldLevel1.strength(), 1.0)
})
```
