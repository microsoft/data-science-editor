# IFrame hosted Jacdac

This tools page can be hosted in HTML IFrame elements in order
to integrate Jacdac into your web site with a minimum of JavaScript.

## Embedded HTML

The following HTML snippet inserts a Jacdac IFrame in the page
(skinning the IFrame is beyond this guide as it is typically web site specific).

-   dashboard

```html copy
<iframe
    style="width: 100%; height: 27rem; border: none;"
    allow="usb;serial;bluetooth"
    sandbox="allow-scripts allow-downloads allow-same-origin"
    aria-label="Jacdac Dashboard"
    src="https://microsoft.github.io/jacdac-docs/dasboard?embed=1"
/>
```

-   data collector

```html copy
<iframe
    style="width: 100%; height: 27rem; border: none;"
    allow="usb;serial;bluetooth"
    sandbox="allow-scripts allow-downloads allow-same-origin"
    aria-label="Jacdac Data Collector"
    src="https://microsoft.github.io/jacdac-docs/tools/collector?embed=1"
/>
```
