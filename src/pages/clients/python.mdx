# Jacdac Python

**Beta: the python layer is still in pre-release mode and beta quality.**

Jacdac provides a Python (3.7+) client package to run programs on Raspberry Pi.
Python projects can also be developed on desktop Python using the developer tools.

-   [GitHub project](https://github.com/microsoft/jacdac-python)
-   [API Reference](https://jacdac-python.readthedocs.io/)

## PyPi Package

Add [jacdac-python](https://pypi.org/project/jacdac-python/) to your project using `pip` (or `pip3` on Windows)

```
python -m pip install --upgrade --user jacdac-python
```

## Connecting to hardware

The transport used to connect to the Jacdac device typically varies a lot depending on the platform and the usage scenario.

### JacHAT

A JacHAT is a Jacdac adapter for the Raspberry Pi that provides an SPI bridge to the Jacdac bus.
