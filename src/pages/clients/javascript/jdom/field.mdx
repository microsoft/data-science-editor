# JDField

A **field** is a [JDOM](/clients/javascript/jdom/) [node](/clients/javascript/jdom/node/) that represents a data field in a [register](/clients/javascript/jdom/register/). 

This object inherits methods and properties from [node](/clients/javascript/jdom/node/).

* [API reference](https://microsoft.github.io/jacdac-ts/classes/JDField.html)

## Methods

### ``name``

The name of the field as defined in the specification.

```typescript
const name = field.name
```

### ``unit``

The unit of the value as defined in the service speciciation.

```typescript
const unit = field.unit
```

### ``value``

The value of the field received in the latest package. If no data has been received, undefined.

```typescript
const v = field.value
```
