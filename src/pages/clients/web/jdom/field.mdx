# JDField

> A **field** is a [JDOM](/clients/web/jdom/) [node](/clients/web/jdom/node/) that represents a data field in a [register](/clients/web/jdom/register/). 

This object inherits methods and properties from [node](/clients/web/jdom/node/).

## Methods

### ``name``

The name of the field as defined in the specification.

```typescript
const name = field.name
```

### ``unit``

The unit of the value as defined in the service speciciation.

```typescript
const unit = field.unit
```

### ``value``

The value of the field received in the latest package. If no data has been received, undefined.

```typescript
const v = field.value
```
